* v2.0.0 (2023-04-30)
  + Code optimizations. Less CPU usage.
  + Get loudness data from all tabs, not only active tab. Quick switch from tab to tab without losing loudness graphs.
  + VU meters mode(6.36+) support (more efficient that using sends to dpmeter or Loudness meter)
  + Autoload/save graph to file. Do not lose current graph when switching projects
  + Selectable curves to select track Ctrl+click to select multiple. Alt+click to clear track's curve.
  + Integrated Loudness evolution display for master track
  + Loudnes Range evolution display for master track
  + Dynamic Range display for master track (experimental)
  + IL, ML, SL, DR for master track, and VU modes(peaks, RMS, LUFS-M, LUFS-S) for tracks.
  + Compare button: right click to save snapshot of the current track loudness curve and left click to enable disable visualization of the snapshot
  + Zoom All button from =START to =END
  + Zoom Loop button 
  + Zoom Lock or unlock for Synchronization with arrange
  + New Settings window for Graph related options
  + Settings: Refresh time. Allows for less graph updates to save CPU for graph display.
  + Settings: Show/Hide Regions and Markers
  + Settings: Filled graph 
  + Settings: Filled graph for Master track
  + Settings: play cursor color, master graph default color.
  + Settings: Show 3 reference levels, custom color, on/off and with over-level color change
  + Settings: Show/hide grid
  + Settings: PrePlay. Positions cursor 3 seconds before clicked on graph, to allow for SL buffer to calculate correct value
  + Double click on graph to play/pause at position
  + Mouse wheel over right scale top and bottom areas, to adjust vertical scale top and bottom (must be in LUFS mode, not EBU modes)
