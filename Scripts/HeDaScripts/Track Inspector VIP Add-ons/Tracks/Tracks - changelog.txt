---------------------------------------------------------------------------------------
 Changelog (What's new)     
---------------------------------------------------------------------------------------
 
* v1.2.1 (2024-08-15)
  # fix glitch when typing in search box.

* v1.2.0 (2024-08-14)
  + In search box, Tab key, or Down arrow Key: switches focus to track list, and selects first track in the filtered list
  + In search box, Enter key: select first track in the filtered list
  + If focus is on track list, Up and Down arrow keys to select tracks from the filtered list
  + If focus is on track list, Tab key to switch focus back to search box
  + panel menu: more options to show/hide individual option buttons

* v1.1.1 (2023-08-26)
  + zoom buttons to change font size (a very small unreadable font size can still be like a map view)
  + Depth level filtering buttons to increase/decrease the folder depth maximum value
  # fix: Tags column alignment to the right
  # fix: some optimizations

* v1.1.0 (2023-08-09)
  + REAPER 7 support

* v1.0.1 (2021-11-22)
  + Do not steal focus for search box when switching pages
  + Focus search box when running the action: HeDa_Track Inspector 2 VIP_Go to panel Tracks

* v1.0.0 (2021-10-08)
  + Search filter (Tab key, or right click to clear search box)
  + Track: Right click to show track menu
  + Panel Menu: Clear and Save sets of favorites
  + Set buttons: left click load set of favorites
  + Set buttons: right click select tracks from set
  + TCP and MCP filter
  + Tags display 
  + Ctrl click to select multiple tracks
  + Shift click to select multiple contiguous tracks, 
  + Panel Menu: show/hide different elements
  + Panel Menu: refresh time to update track list
